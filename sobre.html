Sobre

Esta pagina web foi feita para adquirir nota e conhecimentos na materia de artes, aprendizados de como montar um projeto, e este trabalho é a nossa pratica, este projeto tanto artistico como social, tem o intuito de pessoas que curtem o anime verem musicas feitas pelos próprios fans

Nosso Objetivo com esse projeto é levar o trabalho dos criadores das musicas para pessoas que tem acesso a internet, além de ter como objetivo principal entreter as pessoas que são fans do anime com as musicas
